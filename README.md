## Author 
Michael Appuhn

## Description
Based on screenprints by Anni Albers, I'm making a small oFX app to explore shape patterns in a grid.

From there, I want to open it up to more shapes.

So far in the making:
- Right Triangles
- Isoceles Triangles
- Squares
- Circles
- et cetera
